1. Для запуска приложения выполните метод main класса Application
2. Encoding установлен в UTF-8
3. Приложение работает с базой PostgreSQL (postrges / postgres), каждый раз при запуске создает таблицы (ddl)
4. http://localhost:8080 - стартовая страница
5. http://localhost:8080/login - страница входа/регистрации
6. http://localhost:8080/hello - главная страница Личного кабинета
7. http://localhost:8080/db/all - страница дострупна залогиненому пользователю, оставил, чтобы можно было
посмотреть базу данных.
8. Пароли шифруются с помощью BCcypt
9. При вводе неправильного / пустого / повторяющегося email выводятся соответствующие сообщения на странице login
10. Для тестирования отправки сообщений создан почтовый ящик gmail. Всё забито в application.properties
